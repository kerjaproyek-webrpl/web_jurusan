Tentang Kami

Rekayasa Perangkat Lunak.

SMK Negeri 5 Malang Berdiri pada tahun 1998 di atas tanah seluas 13.816 m2 dengan luas bangunan 3.343 m2 terletak di lokasi strategis dalam wilayah Kota Malang.

Menggunakan Competency Based Training (CBT) dan Production Based Training (PBT), diharapkan setelah mengikuti pendidikan dan pelatihan di SMK Negeri 5 Malang lulusa dapat mengimplementasikan keahlian secara produktif sesuai kompetensi yang dimiliki

  • Website

    Kami handal dalam pembangunan aplikasi berbasis Web, termasuk PHP, HTML, CSS, Javascript dan lainnya

  • Java

    Kami handal dalam pemrograman Java

  • .net

    Kami juga handal dalam pemrograman .Net, dengan Visual Studio atau yang lainnya

  • Basis data

    Kami handal dalam pengolahan Basis Data, termasuk MySql, Oracle, dan lainnya
